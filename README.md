# 🧩 AlgoPlay

**AlgoPlay** is an Android application that transforms classic Data Structures & Algorithms into interactive mobile experiences.

Instead of solving algorithmic problems only on coding platforms, AlgoPlay demonstrates how those algorithms can power real Android applications with modern architecture and intuitive user interfaces.

Each game or tool focuses on a specific algorithm, allowing users to play, explore, and understand how the underlying logic works.

---

## ✨ Vision

> **From LeetCode Problems → Production Android Apps**

The goal of this project is to bridge the gap between algorithmic problem solving and Android application development.

Every feature demonstrates:

* Real-world application of DSA
* Modern Android architecture
* Interactive UI built with Jetpack Compose
* Clean, testable Kotlin code

---

## 🏗️ Architecture

```
AlgoPlay
│
├── app                 // Launcher & Navigation
├── core                // Shared UI, Theme, Utilities
│
├── feature-sudoku
├── feature-word-search
├── feature-maze
├── feature-n-queens
└── ...
```

Each algorithm lives in its own independent feature module.
